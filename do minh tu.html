Giới thiệu HTML và CSS

Giới thiệu HTML

HTML là gì

HTML – tên viết tắt của HyperText Markup Language, ngôn ngữ đánh dấu siêu văn bản, là ngôn ngữ đánh dấu tiêu chuẩn để tạo các trang web và ứng dụng web.

Giới thiệu CSS

CSS là gì

CSS (Cascading Style Sheets) là ngôn ngữ định dạng đặc biệt được dùng để mô tả cách thể hiện của văn bản HTML trong trang web.

Mỗi phần định dạng CSS đều có cú pháp chuẩn là bộ chọn (thuộc tính: giá trị ).
